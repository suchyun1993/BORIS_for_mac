BORIS_for_Mac
=============

Install:
1. Open this disk image.
2. Drag BORIS_for_Mac.app into Applications.
3. Launch the app from Applications.

First launch on macOS:
- Because this build is not signed or notarized yet, macOS may block the first launch.
- If that happens, go to System Settings > Privacy & Security and click "Open Anyway".

Notes:
- This package is the embedded-video macOS variant we customized locally.
- This package is built as a standalone app (no conda environment required).
- FFmpeg should be available via Homebrew (`brew install ffmpeg`) for best compatibility.
